CSS Variables Usage

/* Import Sora from Google Fonts */ @import url('https://fonts.googleapis.com/css2?family=Sora:wght@300;400;500;600;700&display=swap'); /* Component Example */ .kgp-card { font-family: var(--kgp-font-family); /* Sora */ background: var(--kgp-bg-surface); border: 1px solid var(--kgp-border-light); border-radius: var(--kgp-radius-xl); padding: var(--kgp-card-padding); box-shadow: var(--kgp-shadow-sm); transition: all var(--kgp-transition-base); } .kgp-card:hover { transform: translateY(-4px); box-shadow: var(--kgp-shadow-lg); } .kgp-btn-primary { background: var(--kgp-accent); color: var(--kgp-text-on-accent); height: var(--kgp-button-height); border-radius: var(--kgp-radius-md); font-weight: var(--kgp-font-medium); }

KGProDesign 2026 Quick Reference
Available Themes: Urban Gray Pearl (Primary: Slate grays, Accent: Blue #3B82F6)
Still Night Pearl (Primary: Indigo, Accent: Indigo #4F46E5) Naming Convention: All CSS variables use --kgp-* prefix Component Standards: Button Height: 44px (--kgp-button-height)
Input Height: 48px (--kgp-input-height)
Default Border Radius: 8px (--kgp-radius-md)
Card Padding: 20px (--kgp-card-padding)
Icon Size: 24px (--kgp-icon-size) Typography: Font: Sora (Google Fonts) with system fallbacks (--kgp-font-family)
Mono: SF Mono, Monaco, Consolas (--kgp-font-mono)
Base size: 16px
Scale: xs(12), sm(14), base(16), lg(18), xl(20), 2xl(24), 3xl(30), 4xl(36) Transitions: Fast: 150ms (--kgp-transition-fast)
Base: 250ms (--kgp-transition-base)
Slow: 350ms (--kgp-transition-slow) Theme Switching: Set data-theme="urban-gray" or data-theme="still-night" on html element. Set data-mode="light" or data-mode="dark" for mode switching.
