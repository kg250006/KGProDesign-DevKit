<checklist_llc>
LLC FORMATION CHECKLIST

PHASE 1: PRE-FORMATION DECISIONS

Business Planning:
- [ ] Define business purpose and activities
- [ ] Identify all members and ownership percentages
- [ ] Determine initial capital contributions
- [ ] Decide on management structure (member-managed vs. manager-managed)
- [ ] Choose tax treatment (disregarded, partnership, S-corp, C-corp election)

Name Selection:
- [ ] Brainstorm LLC name options
- [ ] Check name availability in formation state
- [ ] Search USPTO for trademark conflicts
- [ ] Check domain name availability
- [ ] Reserve name if needed (varies by state)

State Selection:
- [ ] Determine where business will operate
- [ ] Evaluate home state vs. Delaware vs. Wyoming
- [ ] Understand foreign qualification requirements
- [ ] Calculate total formation and annual costs

PHASE 2: FORMATION FILING

Articles of Organization:
- [ ] Prepare Articles of Organization
- [ ] Include required information:
  - [ ] LLC name
  - [ ] Principal office address
  - [ ] Registered agent name and address
  - [ ] Management structure (if required)
  - [ ] Organizer information
- [ ] File with Secretary of State
- [ ] Pay filing fee
- [ ] Receive stamped/filed copy

Filing Fee Reference:
- Delaware: $90
- California: $70
- Wyoming: $100
- (Verify current fees before filing)

PHASE 3: POST-FORMATION SETUP

Registered Agent:
- [ ] Confirm registered agent is in place
- [ ] Provide registered agent with LLC documents
- [ ] Set up forwarding for legal mail

Operating Agreement:
- [ ] Draft Operating Agreement
- [ ] Include provisions for:
  - [ ] Member information and contributions
  - [ ] Profit/loss allocation
  - [ ] Distribution policy
  - [ ] Management structure
  - [ ] Voting rights
  - [ ] Transfer restrictions
  - [ ] Dissolution procedures
- [ ] Have all members sign
- [ ] Store safely (not filed with state)

EIN (Employer Identification Number):
- [ ] Apply for EIN with IRS (Form SS-4 or online)
- [ ] Receive EIN confirmation letter
- [ ] Required for: bank accounts, hiring, tax filings

Bank Account:
- [ ] Gather required documents:
  - [ ] Articles of Organization
  - [ ] Operating Agreement
  - [ ] EIN confirmation
  - [ ] ID for authorized signers
- [ ] Open business bank account
- [ ] NEVER commingle personal and business funds

PHASE 4: COMPLIANCE SETUP

Foreign Qualification (if applicable):
- [ ] Determine if foreign qualification needed in other states
- [ ] File foreign qualification in each required state
- [ ] Appoint registered agent in each state
- [ ] Track annual requirements for each state

Licenses and Permits:
- [ ] Research required business licenses
- [ ] Apply for state business license
- [ ] Apply for local business license
- [ ] Obtain industry-specific permits

Tax Registration:
- [ ] Register for state taxes
- [ ] Register for sales tax (if selling taxable goods/services)
- [ ] Set up payroll tax accounts (if hiring employees)

PHASE 5: ONGOING COMPLIANCE

Annual Requirements:
- [ ] Calendar annual report deadline
- [ ] Calendar franchise tax payment deadline
- [ ] Calendar registered agent renewal
- [ ] Set up reminders (60, 30, 7 days before)

Record Keeping:
- [ ] Establish record retention system
- [ ] Store:
  - [ ] Formation documents
  - [ ] Operating Agreement and amendments
  - [ ] Meeting minutes and resolutions
  - [ ] Financial records
  - [ ] Tax returns
  - [ ] Contracts and agreements
</checklist_llc>

<checklist_corporation>
CORPORATION FORMATION CHECKLIST

PHASE 1: PRE-FORMATION DECISIONS

Business Planning:
- [ ] Define business purpose
- [ ] Identify incorporator(s)
- [ ] Determine initial directors
- [ ] Plan initial shareholders and share allocation
- [ ] Decide on authorized share structure
- [ ] Choose C-Corp vs. S-Corp tax treatment

Name Selection:
- [ ] Check name availability in formation state
- [ ] Search USPTO for trademark conflicts
- [ ] Verify domain availability
- [ ] Reserve name if needed

State Selection (C-Corp):
- [ ] Default: Delaware for VC-backed startups
- [ ] Alternative: Home state for local businesses
- [ ] Understand franchise tax implications

PHASE 2: FORMATION FILING

Certificate of Incorporation:
- [ ] Prepare Certificate of Incorporation
- [ ] Include:
  - [ ] Corporation name
  - [ ] Registered agent name and address
  - [ ] Authorized shares (number and par value)
  - [ ] Incorporator name and address
  - [ ] Purpose clause
- [ ] File with Secretary of State
- [ ] Pay filing fee
- [ ] Receive filed copy

For Delaware:
- Filing fee based on authorized shares
- Minimum $89 for simple filing
- Consider authorized share strategy to minimize franchise tax

PHASE 3: ORGANIZATIONAL ACTIONS

Initial Board Meeting (Organizational Meeting):
- [ ] Schedule organizational meeting
- [ ] Adopt bylaws
- [ ] Elect officers (President, Secretary, Treasurer at minimum)
- [ ] Authorize issuance of initial shares
- [ ] Approve initial share issuances
- [ ] Designate bank accounts
- [ ] Adopt fiscal year
- [ ] Adopt form of stock certificate
- [ ] Approve initial transactions
- [ ] Document minutes

Bylaws:
- [ ] Draft bylaws covering:
  - [ ] Shareholder meetings
  - [ ] Board meetings
  - [ ] Officer roles
  - [ ] Stock transfer procedures
  - [ ] Indemnification
  - [ ] Amendment procedures

EIN:
- [ ] Apply for EIN with IRS
- [ ] Receive confirmation

Bank Account:
- [ ] Gather required documents
- [ ] Open business bank account

PHASE 4: STOCK ISSUANCE

Stock Documentation:
- [ ] Prepare stock subscription agreements
- [ ] Issue stock certificates (or book entry)
- [ ] Maintain stock ledger
- [ ] Collect payment for shares

Founder Stock Considerations:
- [ ] Consider vesting schedules
- [ ] Prepare restricted stock agreements
- [ ] FILE 83(b) ELECTIONS WITHIN 30 DAYS (critical!)
- [ ] Mail 83(b) via certified mail, keep receipt

S-Corp Election (if applicable):
- [ ] Verify all requirements met:
  - [ ] 100 or fewer shareholders
  - [ ] Only individuals, estates, certain trusts
  - [ ] US citizens/residents only
  - [ ] One class of stock
- [ ] File Form 2553 within 75 days of formation (or by March 15 for current year)
- [ ] All shareholders must sign

PHASE 5: COMPLIANCE SETUP

Delaware Requirements:
- [ ] Pay annual franchise tax by March 1
- [ ] File annual report by March 1
- [ ] Maintain registered agent

Foreign Qualification:
- [ ] Qualify in states where doing business
- [ ] Track annual requirements in each state

Licenses and Permits:
- [ ] Obtain necessary business licenses
- [ ] Industry-specific permits

PHASE 6: ONGOING COMPLIANCE

Annual Requirements:
- [ ] Annual shareholders meeting
- [ ] Annual board meeting
- [ ] File annual reports
- [ ] Pay franchise taxes
- [ ] Maintain corporate records

Corporate Formalities:
- [ ] Document major decisions in minutes/resolutions
- [ ] Maintain separation of personal/corporate finances
- [ ] Use corporation name on all contracts
- [ ] Follow bylaws procedures
</checklist_corporation>

TYPICAL FORMATION TIMELINE

Day 1-2: Pre-formation decisions
Day 3-5: Prepare and file formation documents
Day 5-10: Receive filed documents (varies by state and expedite)
Day 10-15:
- Obtain EIN
- Draft operating agreement/bylaws
- Hold organizational meeting (corporations)
Day 15-30:
- Open bank account
- Initial capital contributions
- Issue stock/membership interests
- File 83(b) elections (corporations with restricted stock)

Time-Sensitive Items:
- 83(b) election: Within 30 days of stock issuance
- S-Corp election: Within 75 days or by March 15
- Foreign qualification: Before conducting business in state
